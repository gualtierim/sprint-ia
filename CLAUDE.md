# Sprint-IA — Istruzioni di progetto

## Contesto obbligatorio

All'inizio di ogni conversazione, leggi SEMPRE questi file prima di rispondere a qualsiasi richiesta:

- `.cursor/skills/sprint/SKILL.md` — Organizzazione meta-repo e sviluppo
- `.cursor/skills/sprint-api/SKILL.md` — Workflow contract-first API (OpenAPI)
- `.cursor/skills/schema/SKILL.md` — Gestione schema DB, migrazioni e accesso dati
- `.cursor/rules/frontend-http.mdc` — Convenzioni HTTP frontend Angular
- `.cursor/skills/table-columns/SKILL.md` — Colonne tabella configurabili (popup + sessionStorage)
- `.cursor/skills/backend/SKILL.md` — Backend sprintbff: MapStruct, Manager, conversioni Row→VO

Queste sono le regole di progetto. Rispettale in ogni intervento sul codice.

## Comandi disponibili

Le stesse skill sono invocabili singolarmente come comandi:

- `/commands/sprint`
- `/commands/sprint-api`
- `/commands/schema`
- `/commands/table-columns`
- `/commands/backend`
